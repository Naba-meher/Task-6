# Task-6
# Interactive Sales Dashboard

# Project Overview
This project builds an interactive sales dashboard that analyzes a sales dataset to reveal trends, product performance, and customer segmentation. It uses **Seaborn** for polished statistical plots and **Plotly** for interactive exploration. The dashboard is designed to be clear, reproducible, and presentation-ready.

# Key Features
- Static visualizations with Seaborn: bar chart, line chart, boxplot, violin plot, heatmap  
- Interactive visualizations with Plotly: scatter, sunburst, interactive time series  
- Data validation and preprocessing with pandas  
- Exports static PNGs and interactive HTML files for sharing
